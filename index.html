Learning SQLite?

Practice with a containerized SQLite CLI environment in 5 simple steps:

  1. Make sure Docker is running
  2. From VSCode (using the "Remote - Containers" extension), "Open Folder In Container".
  3. Assuming you already copied or git clone'd the code in this GitHub repo, select this folder
  4. You can check Docker to make sure the container is running (it might take a minute or so to spin up)
  5. From inside VSCode's command prompt (running *inside* the container), start the SQLite CLI with command sqlite3 test.db
